X-Vortex-Token Setup Guide

  1. Open the webhook settings page in VORTEX Portal.
  2. Set the webhook URL to https://webhook.vivotek.tools/webhook.
  3. Add a custom header named X-Vortex-Token.
  4. Set the header value to the token saved in this dashboard.
  5. After saving, this server accepts only events whose X-Vortex-Token header matches the current token.
